# Tic‑Tac‑Toe Web App

Eine einfache, moderne und responsive Tic‑Tac‑Toe Web‑App, entwickelt mit HTML, CSS und JavaScript.  
Das Projekt dient als Übung für DOM‑Manipulation, Event‑Handling und grundlegende Game‑Logic im Browser.



## Features

- ✔️ Klassisches Tic‑Tac‑Toe (Spieler X gegen Spieler O)  
- ✔️ Visuelles Highlighting der Gewinnerkombination  
- ✔️ Reset‑Button zum Neustarten des Spiels  
- ✔️ Responsive UI – funktioniert auf Desktop und Mobile  
- ✔️ Saubere, gut strukturierte Game‑Logik  



##  Technologien

- **HTML5** – Struktur der App  
- **CSS3** – Layout, Styling, Responsive Design  
- **JavaScript** – Spiellogik, DOM‑Interaktionen
